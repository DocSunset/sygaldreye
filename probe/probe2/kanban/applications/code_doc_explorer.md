# Code / research / document viewer, explorer, annotator + hypertext editor (brainstorm 2026-06-23)

Application note, not yet a work item.

Use the graph as a spatial workspace for reading and authoring: view
and explore codebases, research papers, and documents; annotate them;
and link them into navigable hypertext. Documents and their fragments
become wirable graph content — wire a code symbol, a citation, an
annotation into the rest of the patch. The authoring/linking side is
the hypertext editor for the pedagogical hypertexts in
[[web_ui_hypertext]] (and shares the Xanadu lineage noted in
[[social_appliverse]]).

Open questions: text/document rendering surface in-VR, how annotations
and links are stored (transclusion vs reference), and how code
intelligence (symbols, xrefs) feeds the graph.
