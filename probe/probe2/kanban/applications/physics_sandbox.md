# Physics sandbox (brainstorm 2026-06-23)

Application note, not yet a work item.

Use the graph as a VR physics playground — spawn bodies, forces,
constraints, and fields as wirable nodes and play with them in-room.
Tweak parameters live, wire physics state into sound/visuals, build
contraptions by hand. Pedagogical sibling to the game engine note:
here the physics IS the content, not a means to a game.

Open questions: solver/engine (existing physics blocks vs a real rigid
body lib), interaction model (grab/throw with XR hands), and how much
fidelity vs playfulness.
