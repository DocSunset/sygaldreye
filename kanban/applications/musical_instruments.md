# Musical instruments (brainstorm 2026-06-24)

Application note, not yet a work item.

Use the graph to build playable musical instruments — synthesis,
sampling, sequencing, and effects as wirable audio nodes, with the
instrument's body and controls authored spatially in-VR. Reach out and
play it; design new interfaces that only exist in 3D. The audio engine
(block/region signal flow, spatialized sources) is already the
substrate; this frames it as lutherie.

Open questions: low-latency playability (audio + hand tracking jitter),
mapping XR gesture to sound, and tuning/scale systems.
